# AttendanceSystem

This is a simple Java project that prints a welcome message.

## How to Compile

From the project root directory, run:

```
javac src/com/school/Main.java
```

## How to Run

From the project root directory, run:

```
java -cp src com.school.Main
```

├── src/
│   └── com/
